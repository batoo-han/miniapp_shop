# Руководство пользователя

## Администратор

### Вход в админку

1. Откройте `/admin/` в браузере.
2. Введите логин и пароль (из конфигурации).
3. После успешного входа получите JWT, сессия сохраняется.

### Список товаров

- **Панель статистики**: всего товаров, опубликовано, просмотры.
- **Фильтры**: поиск по названию/артикулу, категория, производитель, статус (все / опубликованные / черновики). Кнопки «Применить» и «Сбросить».
- **Таблица**: превью, название, артикул, цена, производитель, категория, статус, просмотры. Клик по строке раскрывает варианты (цвет/размер, остатки).
- **Пагинация** при большом количестве товаров.

### Добавление товара

1. Нажмите «Добавить товар» в хедере.
2. Заполните: slug, название, артикул, производитель, категорию (если созданы).
3. Краткое и полное описание.
4. Цена (опционально).
5. Загрузите фото (кнопки ↑/↓ для сортировки).
6. Прикрепите файлы (PDF, инструкции).
7. Добавьте ТТХ (кнопка «Добавить ТТХ»).
8. Добавьте варианты (цвет, размер и т.п.) с остатками.
9. Включите «Опубликован» для отображения в витрине.
10. Сохраните.

### Редактирование и удаление

- Список: «Редактировать» — переход в форму. «Удалить» — с подтверждением.
- В форме: все поля, ТТХ (добавить/удалить), варианты (редактировать остатки, удалить), фото (сортировка, удаление).

## Пользователь витрины (Mini App)

1. Откройте Mini App через бота в Telegram.
2. Список — плитка с фото, названием, ценой и кратким описанием.
3. Клик по карточке — переход в карточку товара.
4. В карточке: галерея фото, описание, ТТХ, кнопки скачивания файлов.
5. «Связаться со специалистом» — переход в Telegram (бот или чат).
